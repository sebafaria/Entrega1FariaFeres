# Entrega1FariaFeres

Este repositorio corresponde a la entrega intermedia del proyecto final, del curso de CODERHOUSE, comisión 36945
Este trabajo fue desarrollado por: 
    - Faria Sebastian
    - Feres Samir

En primer lugar, este desarrollo fue realizado para una constructora que tenía la necesidad de contar con una web donde puedan cargar obras, clientes y empleados e interactuar con estos datos. 

A continuación se detallan las funcionalidades de esta web y como utilizarlas. 

INSERTAR DATOS: Este formulario nos permite generar nuevas obras. Se accede a traves de obras/crear_obra

BUSCAR DATOS: A traves de este formulario, accesible desde obras/search_obras se pueden buscar con el numero de cliente, las obras que la constructora realizo para el mismo. 

LISTADO DE OBRAS: Con esta funcionalidad a la cual se accede a trabes del URL obras/listar_obras/ podemos ver un listado completo de las obras de la constructora, tanto en proceso como ya finalizadas. Esta funcionalidad utiliza una herencia HTML como se puede ver en los códigos.
